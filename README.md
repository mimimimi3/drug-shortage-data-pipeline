# FDA Drug Shortage ELT Pipeline

End-to-end auotmated ELT pipeline ingesting daily FDA drug shortage data from the openFDA REST API, storing raw data in Google Cloud Storage, loading into BigQuery, transforming with dbt, orchestrated by Airflow, containerized with Docker, and 
provisioned with Terraform. Visualized with Looker Studio.

## Tech Stack
Python | Apache Airflow | dbt | BigQuery | Google Cloud Storage | Docker | Terraform | Looker Studio
